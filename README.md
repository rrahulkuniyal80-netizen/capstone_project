# 🏡 Real Estate Price Prediction & Recommendation System

A comprehensive Machine Learning project that predicts real estate prices and recommends similar properties based on user preferences. This project includes end-to-end data preprocessing, exploratory data analysis, feature engineering, model building, and a recommendation system.

---

## 📌 Project Overview

The goal of this project is to help users estimate property prices and discover similar properties using machine learning techniques.

The project covers the complete ML workflow:

- Data Collection
- Data Cleaning
- Exploratory Data Analysis (EDA)
- Feature Engineering
- Feature Selection
- Model Training & Evaluation
- Property Recommendation System

## 📊 Dataset

The dataset contains residential property information from Gurgaon, including:

- Property Type
- Sector
- Area
- Number of Bedrooms
- Bathrooms
- Balconies
- Furnishing Status
- Floor Information
- Property Age
- Price
- Price per Sq.ft.
- Amenities
